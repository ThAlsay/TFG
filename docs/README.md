# Hephaestus

El proyecto Hephaestus es el resultado del Trabajo de Fin de Grado del autor. Es un motor de videojuegos distribuidos escrito enteramente en Elixir.

Un motor de viedeojuegos puede estar especializado en un género o tipo de videojuego concreto, o ser generalista y abarcar un amplio abanico de posibilidades.

Haephaestus es un motor especializado en el género RPG y que cuenta con la capacidad de poder ser utilizado para crear videojuegos que cumplan con una
arquitectura distribuida. Este último apartado es opcional y también se pueden crear videojuegos que se ejecuten de manera local, sin seguir una arquitectura
distribuida.

Para desarrollar con este motor es importante tener nociones de programación distribuida en Elixir empleando la librería _GenServer_ incluida en el propio
lenguaje.
